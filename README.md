# Expense Tracker App

A modern expense tracker built using **React (JSX)** and **Tailwind CSS** that helps users track income and expenses, view monthly summaries, and persist data using localStorage.

---

## Features

- Add income and expense entries
- Delete individual entries
- Income vs Expense toggle
- Automatic balance calculation
- Monthly expense & income summary
- Date-based tracking
- Data persistence using localStorage
- Responsive, mobile-first UI
